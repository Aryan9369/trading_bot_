# Binance Futures Trading Bot CLI

A production-quality CLI application to interact with the Binance Futures Testnet (USDT-M). Built with Python and `python-binance`.

## Features
- Place **MARKET** and **LIMIT** orders.
- Supports **BUY** and **SELL** sides.
- Robust input validation.
- Comprehensive logging to `bot.log`.
- Error handling for API and Network issues.
